Sort Multiple Products at once in WooCommerce using drag and drop.

== Description ==
You can select multiple products under products=>sort and select multiple products clicking on products with shift/ctrl/cmd key and drag and drop them as you used to do single product

== Installation ==
1. Upload the folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the “Plugins” menu in WordPress.
3. Yeah! it\'s that simple
